One-time setup

1

Fork this repository

Go to GitHub and click Fork — this gives you your own free copy.

2

Create a GitHub Personal Access Token

Go to GitHub → Settings → Tokens, tick repo scope, and paste it below. It lets this page save your booking config.

3

Fill in the form below and click Schedule

GitHub Actions will wake up at exactly the right time and book the shelter for you. 100% free.
